Belajar Struktur dan Tag Pada HTML

Willy Zondri

Pada tulisan sebelumnya telah dijelaskan tentang HTML dasar. Di tulisan kali ini kita akan belajar tentang Struktur Dasar dari penulisan HTML.

Struktur Dasar HTML

Secara umum pada setiap bahasa script maupun bahasa pemrograman akan ada struktur dasar yang membangun script maupun bahasa pemrograman.

Didalam HTML struktur dasar penyusunnya terdiri dari tag : <html>, <head>, dan </body>.

Berikut ini adalah contoh kodingan dari Struktur dasar HTML.

<!DOCTYPE html>
<html>
  <head>
    <title>Struktur Dasar dan Tag Pada HTML</title>
  </head>
  <body>
    <h1>Struktur dan Tag HTML</h1>
    <p>Belajar Struktur dan Tag HTML</p>
  </body>
</html>

Tag HTML

Sebelum menuliskan tag html terlebih dahulu kita mendeklarasikan tag <!DOCTYPE html> yang berguna untuk memberitahu browser bahwa halaman yang sedang dimuat / diproses adalah sebuah halaman HTML.

Setelah itu barulah kita menuliskan tag HTML yang berfungsi untuk membungkus keseluruhan isi dari halaman HTML.

Tag HEAD

Tag head adalah tag yang dituliskan sesudah tag html. Tag head berfungsi untuk menampung definisi halaman seperti Judul Halaman, Stylesheet (CSS), Script Javascript dan banyak lagi lainnya.

Setiap definisi yang berada didalam tag head tidak akan ditampilkan kedalam browser pengguna website / halaman web.

Tag Body

Tag Body adalah tag yang ditulis setelah tag head. Tag ini berguna untuk menampung isi dari halaman website dan ditampilkan pada web browser.

Pada tag ini biasanya tag – tag yang ada didalam html ditampung untuk membuat sebuah halaman website dapat dilihat dan dibaca oleh pengguna website.

Tag tag Pada HTML

Pada paragraf diatas telah dijelaskan tentang tag penyusun HTML dasar, selanjutnya ialah tag – tag yang dapat digunakan didalam tag body HTML.

Berikut adalah tag – tag tersebut:

  • h1 – h6 : Heading
  • p : Paragraf
  • br : Membuat satu baris putus
  • hr : Membuat garis horizontal
  • <!– .. –> : Membuat komentar pada html
  • div, span : Membuat sebuah bagian dalam dokumen
  • article : Article (Tag HTML5)
  • table : Membuat tabel
  • form : Membuat from
  • i : Tulisan miring
  • b : Tulisan cetak tebal
  • ul, ol : Membuat list
  • li : Isi dalam list

Itulah beberapa tag yang sering digunakan didalam pembuatan sebuah halaman HTML.
