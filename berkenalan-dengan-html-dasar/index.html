Berkenalan Dengan HTML Dasar

Willy Zondri

HTML merupakan singkatan dari Hyper Text Markup Language. HTML adalah bahasa markup yang digunakan untuk membangun sebuah halaman web. HTML inilah yang menjadi kerangka penyusun halaman web seperti yang kita lihat pada laman Browser.

Didalam membangun sebuah halaman website ada Bahasa lain yang menjadi pelengkap HTML, yaitu Javascript dan CSS. CSS berfungsi untuk mempercantik dan memperindah tampilan halaman dan Javascript berfungsi untuk membuat halaman website menjadi interaktif.

Perlu diketahui HTML bukanlah sebuah Bahasa pemrograman melainkan adalah sebuah Bahasa markup yang hanya berfungsi untuk menyusun kerangka laman website dan tidak dapat digunakan untuk menjalankan sebuah fungsi atau prosedur logika.

Sejarah singkat HTML

HTML adalah Bahasa yang dikembangkan pada awal tahun 1980an oleh Tim Berners-Lee yang bekerja disebuah perusahaan (CERN). CERN sendiri merupakan perusahaan yang bergerak dibidang Fisika Nuklir.

Namun dikarenakan banyaknya dokumen penelitian, para peneliti membutuhkan alat yang bisa digunakan untuk berbagi dokumen.

Berdasarkan masalah yang dihadapi, ditahun 1989 Tim Berners mencoba untuk mengembangkan sebuah Bahasa pengolahan teks yang berbasis internet, yang menjadi cikal bakal dari Bahasa HTML dan diperkenalkan kepublik pada tahun 1991.

Element, Tag, dan Atribut HTML

Setelah mengenal sejarah dan HTML secara umum Langkah selanjutnya adalah mempelajari tentang element, tag, dan atribut yang terdapat didalam html.

Tag HTML

Didalam HTML element digunakan untuk membuat struktur pada halaman web dibangun. Element HTML terdiri dari tag pembuka, isi, dan tag penutup. Tag berfungsi sebagai tanda untuk membedakan satu elemen dengan elemen yang lainnya.

Didalam pembuatan elemen html dibutuhkan dua buah tag yaitu tag pembuka dan tag penutup. Diantara tag pembuka dan penutup terdapat isi / konten dari elemen HTML itu sendiri. Berikut ini adalah contoh penulisan tag sebuah elemen html:

<html>Isi Elemen HTML</html>

Dalam penulisan tag HTML bisa ditulis menggunakan huruf besar maupun kecil, karena rata – rata browser pada saat ini sudah dapat mengenalinya.

Atribut

Setelah mempelajari dan mengenali tentang cara penggunaan tag, tahapan selanjutnya adalah mempelajari tentang Atribut. Atribut sendiri adalah suatu karakteristik tambahan pada TAG HTML. Penulisan atribut terdiri dari beberapa cara diantaranya :

  • Penamaan atribut diikuti oleh tanda = (sama dengan) setelahnya.
  • Nilai dari atribut berada diantara tanda petik (“/’).
  • Jika terdapat lebih dari satu atribut maka dapat dipisahkan dengan menggunakan SPASI.

Berikut adalah contoh penggunaan atribut didalam element HTML:

<div id="nilaiatribut">Isi Element</div>

Menulis Script Pertama HTML

Ditahapan ini akan mempelajari tentang cara penulisan Script HTML pertama. Sebelum memulai penulisan Script HTML ada beberapa perangkat atau tools yang dibutuhkan yaitu :

Teks Editor

Yang pertama text editor. Text editor berguna untuk mengolah kata / script dari html. Buka text editor kemudian pada bagian save as pilih tipe HTML. Untuk text editor sendiri disarankan untuk menggunakan Sublime Text.

Web Browser

Yang kedua web browser. Web Browser berguna untuk mengeksekusi Script HTML yang telah dibuat di text editor sebelumnya. Didalam web browser inilah kita dapat melihat hasil dari Script / Kode dari HTML yang telah kita buat.

Script HTML

Yang ketiga adalah script HTML itu sendiri. Berikut ini adalah contoh dari script html-nya.

<!DOCTYPE html>
<html>
  <head>
    <title>HTML DASAR</title>
  </head>
  <body>
    <h1>HTML DASAR</h1>
    <p class="htmlDasar">Hallo ini adalah tulisan pertama saya menggunakan HTML</p>
  </body>
</html>

Berikut ini adalah hasil script HTML yang dijalankan pada web browser
