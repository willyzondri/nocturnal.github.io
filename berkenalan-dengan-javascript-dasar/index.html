Berkenalan Dengan Javascript Dasar

Willy Zondri

Hampir sama dengan tulisan – tulisan sebelumnya, topik pembahasan yang dibahas tidak jauh – jauh dari kata “Berkenalan”. Didalam mempelajari suatu hal yang baru sebaiknya kita harus mengetahui latar belakang dari hal yang akan dipelajari. Supaya dapat mengerti dan memahami tujuan dari mempejari bahasa ini.

Seperti yang sudah disinggung ditulisan sebelumnya Javascript merupakan salah satu dari tiga komponen / bahasa yang digunakan untuk membangun halaman web bersama dengan HTML dan CSS. Secara umum Javascript berfungsi untuk membuat situs website menjadi lebih interaktif ketika berinteraksi dengan Pengguna. Salah satunya yaitu manipulasi DOM pada halaman HTML.

Sejarah Javascript

Javascript adalah bahasa pemrograman yang dibuat oleh Brandon Eich pada tahun 1995 ketika dia bekerja sebagai programmer di Netscape. Pada awalnya Javascript bernama Mocha kemudian diganti menjadi LiveScript hingga menjadi Javascript.

Penggunaan nama kata Java didalam Javascript digunakan setelah Netscape bekerja sama dengan Sun Microsystem (ORACLE). Penggunaan nama Java sendiri tidak terlepas dari alasan strategi marketing, yang mana strategi marketing ini cukup berhasil melambungkan nama Javascript dikalangan WEB Developer kala itu.

Tak lama setelah Javascript dikembangkan Microsoft meluncurkan bahasa pemrograman baru yang bernama JScript yang merupakan hasil adopsi dari Javascript sendiri dan digunakan untuk Browser IE buatan microsoft.

Di tahun 1997 Netscape melakukan standarisasi Javascript pada ECMA Internasional dan dikenal dengan nama ECMA Script dan terus berkembang hingga saat ini.

Bagaimana Javascript Bekerja

Javascript merupakan bahasa pemrograman yang bekerja pada sisi client (Client Side). Dimana script yang ada pada Javascript diunduh terlebih dahulu oleh Browser / Client baru kemudian diproses / dieksekusi.

Berbeda dengan bahasa pemrograman yang bekerja pada sisi server (Server Side). Bahasa pemrograman dieksekusi terlebih dahulu didalam server baru kemudian diberikan hasilnya pada Client.

Codingan Pertama Javascript

Nah disini akan menjeleaskan tentang langkah – langkah yang dilewati ketika akan menulis Javascript untuk pertama kalinya. Tentu saja didalam dunia perkodingan ada beberapa peralatan yang harus kita siapkan terlebih dahulu.

Text Editor

Teks editor merupakan sebuah aplikasi pengolah kata plain text yang berguna untuk melakukan pengolahan syntax – syntax yang terdapat didalam bahasa pemrograman.

Jika ditelusuri pada mesin pencari banyak sekali ditemui aplikasi / tools pengolah kata (text) ini, contohnya seperti Visual Studio Code, PhpStorm, Bracket, Sublime Text dan masih banyak lagi yang lainnya.

Dari beberapa contoh aplikasi pengolah kata diatas disarankan untuk menggunakan Sublime Text dikarenakan termasuk aplikasi yang cukup ringan.

Web Browser

Web browser adalah suatu aplikasi yang berguna untuk menjalankan Kodingan Javascript yang telah kita buat sebelumnya. Web Browser biasanya digunakan untuk menjelajah halaman Website oleh pengguna reguler CMIW. Disini disarankan untuk menggunakan browser Google Chrome atau Firefox.

Pengetahuan Dasar

Pengetahuan dasar ini maksudnya adalah cara penulisan javascript. Hampir sama dengan CSS, cara penulisan Javascript terbagi menjadi dua cara yaitu:

  • Penyematan : Script Javascript dapat disematkan didalam element HTML dengan menuliskan tag <script> … Kodingan … </script>
  • File terpisah : Script Javascript ditulis didalam file yang terpisah dari halaman HTML dengan ekstensi *.js. Dengan menggunakan tag <script src=”../namafile.js”></script>

Jika peralatan yang dibutuhkan sudah terpenuhi langkah selanjutnya adalah menulis kodingan pertama dengan Javascript. Berikut adalah contoh dari kodingan Javascript.

<!DOCTYPE html>
<html>
  <head>
    <title>Berkenalan dengan Javascript</title>
  </head>
  <body>
    <script type="text/javascript">
      document.write("Halo Ini Tulisan Pertama Dengan Javascript");
    </script>
  </body>
</html>
