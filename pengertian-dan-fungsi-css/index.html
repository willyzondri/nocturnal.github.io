Berkenalan dengan CSS, Pengertian dan Fungsinya

Willy Zondri

Pada tulisan sebelumnya telah membahas tentang pengenalan HTML Dasar. Pada tulisan kali ini akan membahas tentang CSS (Cascade Style Sheet). Seperti yang telah dijelaskan pada tulisan sebelumnya didalam membuat halaman website setidaknya dibutuhkan tiga bahasa pemrograman, yaitu HTML, CSS, dan Javascript.

Ditulisan ini akan mengupas tentang CSS Pengertian dan Fungsi dari CSS itu sendiri. Sebelum mempelajari CSS lebih jauh ada baiknya kita untuk mengenal sejarah CSS terlebih dahulu, seperti kata orang – orang “tak kenal maka tak sayang”.

Sejarah CSS

Sejarah CSS sendiri tidak terlepas dari diciptakannya HTML. Kehadiran HTML memicu untuk hadirnya CSS sendiri. Semua berawal dari SGML (Standard Generalized Markup Language). Seperti yang telah dijelaskan pada tulisan sebelumnya dimana kehadiran SGML inilah yang akhirnya menjadi cikal bakal dari HTML.

Karena tampilan dari HTML sendiri ini terlihat monoton maka dibuatlah sebuah CSS untuk memperindah tampilan dari HTML, ide ini dikemukakan oleh Hakon Wium Lie dengan judul proposal CHSS (Cascading HTML Style Sheet). CSS ini secara resmi diperkenalkan kepada publik pada tahun 1995 saat konferensi W3C (World Wide Web Consortium).

Versi CSS

Pada awal pengembangannya CSS hanya mendukung beberapa pengaturan untuk Element HTML, diantaranya : Margin, Border, Text, Text Color, Table, Padding, dan Image. CSS terus berkembang menjadi versi 2 dan versi 3.

  • CSS Versi 2 tentu saja pengembangan dari CSS versi awal (1) dimana pada versi dua ini dibangun agar format dokumen HTML dapat ditampilkan pada printer.
  • CSS Versi 3 adalah versi penyempurnaan dari CSS versi 1 dan Versi 2, dimana pada versi ini tidak hanya mendukung tampilan printer tapi juga untuk membuat tampilan website menjadi lebih menarik. Dimana pada CSS versi 3 mendukung halaman website untuk diberikan animasi.

Fungsi CSS

Untuk fungsi CSS ini sendiri secara tidak langsung telah dijelaskan pada paragraf – paragraf sebelumnya, dimana CSS ini sendiri befungsi untuk membuat tampilan website terlihat lebih menarik.

Macam – Macam CSS

Kehadiran css tidak hanya memberikan kemudahan kepada pengembang HTML saja, namun juga memberikan keuntungan pada pengguna website dari segi tampilan dimana CSS dapat membuat tampilan Website menjadi terstruktur. Berikut ini adalah macam – macam gaya penulisan CSS:

Embedded CSS

Embedded CSS adalah script css yang ditulis langsung didalam Tag <head> …css… </head> html seperti contoh berikut.

<head>
  <style type="text/css">
    p {
      color: red;
    }
  </style>
</head>

Inline CSS

Inline CSS script css yang dituliskan secara eksplisit pada tag HTML / menjadi sebuah atribut dari tag HTML seperti berikut.

<body>
  <p style="color:red"></p>
</body>

Eksternal CSS

Eksternal CSS script css yang dituliskan diluar halaman / file html dengan format file (.css). Kemudian css ini dipanggil dengan menggunakan tag <link/> HTML. Berikut ini contoh penggunaan Eksternal CSS.

<link rel="stylesheet" type="text/css" href="namafile.css">

Struktur CSS

Hampir sama dengan HTML, CSS juga memiliki struktur dasar yang terdiri dari selector, property, dan value:

  • Selector adalah struktur dari css yang berguna untuk memilih tag HTML mana yang akan diberikan nilai dari CSS itu sendiri. Selector ini dapat memilih HTML berdasarkan TAG, atau berdasarkan atribut TAG HTML yaitu atribut ID dan CLASS. Berikut contoh dari selector.
<!DOCTYPE html>
<html>
<head>
  <title>Struktur CSS</title>
  <style type="text/css">
    p { color: red; }
    #iniID { color: yellow; }
    .iniClass { color: green; }
  </style>
</head>
<body>
  <p>Menggunakan Tag HTML</p>
  <p id="iniID">Menggunakan Atribut ID Tag HTML</p>
  <p class="iniClass">Menggunakan Atribut Class Tag HTML</p>
</body>
</html>
  • Property yaitu atribut untuk mendefinisikan selector.
  • Value adalah nilai dari sebuah property.
