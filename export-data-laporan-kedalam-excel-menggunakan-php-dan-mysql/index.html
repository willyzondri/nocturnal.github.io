Export Data Laporan Kedalam Excel Menggunakan PHP dan MYSQL

Willy Zondri

Laporan merupakan sebuah kumpulan data yang apabila diolah akan menjadi sebuah informasi yang dapat dimanfaatkan untuk berbagai macam kebutuhan seperti bahan rujukan, informasi penjualan, bahkan bisa digunakan untuk memprediksi penjualan dimasa yang akan datang.

Laporan tersebut dapat disajikan dalam berbagai bentuk ekstensi aplikasi pengolah kata seperti PDF, EXCEL, WORD dan lainnya. Pada kesempatan ini kita akan membahas tentang cara mengeksport data laporan kedalam bentuk ekstensi excel menggunakan PHP dan MYSQL.

Sebelumnya ada beberapa persiapan yang harus dilakukan terlebih dahulu yaitu:
1. Membuat database
2. Mengkoneksikan database MYSQL dengan PHP
3. Menampilkan data
4. Dan terakhir mengeksport data kedalam bentuk file excel aka .exl

1. Membuat database

Pada section ini kita akan membuat sebuah database misalkan kita mempunyai sebuah database universitas yang terdiri dari sebuah table mahasiswa terdiri dari atribut nim, nama, email, dan alamat.

2. Mengkoneksikan Database MYSQL dengan PHP

Setelah membuat database selanjutnya adalah mengkoneksikan mysql dengan php. Pada kode editor buatlah sebuah projek universitas dengan sebuah nama file index.php. Dan isilah file index.php dengan kode berikut ini.

<?php 
$conn = mysqli_connect("localhost","root","","universitas");
if (mysqli_connect_errno()){
   echo "Koneksi Belum Berhasil : " . mysqli_connect_error();
}
?>

3. Menampilkan data dari database

Setelah koneksi kedalam database berhasil, langkah selanjutnya adalah menampilkan data dari database kedalam bentuk table html. Berikut ini adalah source codenya.

<?php
  $conn = mysqli_connect("localhost", "root", "", "universitas");
  if (mysqli_connect_errno())
  {
   echo "Koneksi Belum Berhasil : " . mysqli_connect_error();
  }
  ?>
<h3>Data Mahasiswa</h3>
<table border="1" class="table">
  <tr>
    <th>No</th>
    <th>Nim</th>
    <th>Nama</th>
    <th>Email</th>
  </tr>
  <?php
    $query_mysql = mysqli_query($conn, "SELECT * FROM mahasiswa") or die(mysql_error());
    $nomor = 1;
    while ($data = mysqli_fetch_array($query_mysql))
    {
    ?>
  <tr>
    <td><?php echo $nomor++; ?></td>
    <td><?php echo $data['nim']; ?></td>
    <td><?php echo $data['nama']; ?></td>
    <td><?php echo $data['email']; ?></td>
  </tr>
  <?php
    } ?>
</table>

4. Mengekspor data kedalam format excel

Setelah berhasil menampilkan data dalam bentuk HTML selanjutnya adalah melakukan pengeksportan data kedalam format excel dengan menambahkan beberapa baris kode berikut ini diatas kode yang lainnya.

<?php
header("Content-type: application/vnd-ms-excel");
header("Content-Disposition: attachment; filename=Data Mahasiswa.xls");
?>

Menjadi seperti berikut ini.

<?php
  header("Content-type: application/vnd-ms-excel");
  header("Content-Disposition: attachment; filename=Data Mahasiswa.xls");
  $conn = mysqli_connect("localhost", "root", "", "universitas");
  if (mysqli_connect_errno())
  {
    echo "Koneksi Belum Berhasil : " . mysqli_connect_error();
  }
  ?>
<h3>Data Pengguna</h3>
<table border="1" class="table">
  <tr>
    <th>No</th>
    <th>Nim</th>
    <th>Nama</th>
    <th>Email</th>
  </tr>
  <?php
    $query_mysql = mysqli_query($conn, "SELECT * FROM mahasiswa") or die(mysql_error());
    $nomor = 1;
    while ($data = mysqli_fetch_array($query_mysql)) {
    ?>
  <tr>
    <td><?php echo $nomor++; ?></td>
    <td><?php echo $data['nim']; ?></td>
    <td><?php echo $data['nama']; ?></td>
    <td><?php echo $data['email']; ?></td>
  </tr>
  <?php } ?>
</table>
